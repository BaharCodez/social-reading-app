# Design inspiration

Drop reference images here (screenshots, palettes, illustrations, moodboards)
and I'll use them to tune the app's themes and cozy bookshop / botanical vibe.

Helpful if you can:

- **Name files by intent** when easy, e.g. `meadow-palette.png`, `cabin-vibe.jpg`,
  `login-idea.png`. Otherwise any name is fine — just tell me which is which.
- **Group by theme** if you have several (Meadow / Sepia / Cabin / Twilight /
  Forest), or note "this is the overall vibe."
- **Add a line below** about what you like in each (the colours, the mood, the
  layout, the texture…).

Any image format works (PNG/JPG/WEBP/HEIC). After you add them, just say
"check the folder" and I'll look.
